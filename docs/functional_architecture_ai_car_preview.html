STM32 智能小车功能架构图：循迹 + 蓝牙遥控

可编辑源文件：docs/functional_architecture_ai_car.drawio。这张图按小车功能而不是通信链路组织，用来说明系统边界。

小车主要分为两条业务线：循迹模式 和 蓝牙遥控模式。两者共用菜单、按键、安全停车、OLED、电机驱动和底盘执行层。

主功能 A：循迹模式

输入感知：8 路红外循迹传感器，输出黑线状态和 line_err。
姿态辅助：JY61P 通过 USART3 输出 yaw 角度。
LINE_TASK 状态机：处理顺/逆时针循迹、指定角点计数、丢线、转向和完成判断。
输出：根据循迹误差和姿态修正得到左右电机 PWM。

主功能 B：蓝牙遥控模式

手机蓝牙串口：Phone BT，文本发送 F、F5、S0。
语音蓝牙链路：ESP32 -> JDY34 -> JDY31，发送 F5/B5/L5/R5/S0。
USART1 接收：JDY31 TXD -> STM32 PA10，9600 8N1。
BT_REMOTE：解析 F/B/L/R/S 和速度档位，维护 Cmd、Raw、left_pwm、right_pwm，并做超时停车。

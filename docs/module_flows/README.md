# AI Car 单页模块流程图

`docs/module_flow_diagrams_ai_car.drawio` 是多页面 draw.io 文件，但部分预览器或打开方式只显示第一页。为了稳定查看，这里把 4 个模块拆成独立单页文件。

| 文件 | 内容 |
| --- | --- |
| `01_esp32_voice_command_flow.drawio` | ESP32 语音指令生成与发送流程 |
| `02_jdy34_jdy31_ble_link_flow.drawio` | JDY34 / JDY31 蓝牙透传流程 |
| `03_stm32_bt_remote_flow.drawio` | STM32 蓝牙接收、显示与电机控制流程 |
| `04_debug_validation_flow.drawio` | 蓝牙链路分段调试验证流程 |

建议直接用 diagrams.net/draw.io 分别打开这些单页文件。
